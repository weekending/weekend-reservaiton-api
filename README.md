# Reservation API 🦀

### 💻 개발 환경

- Rust: 1.87.0
- Cargo: 1.87.0
- Axum: 0.8.4

<br>

# 1. 로컬 환경설정

## 1.1 코드 Clone

```sh
$> git clone https://github.com/weekending/weekend-reservaiton-api.git
$> cd weekend-reservaiton-api
```

## 1.2 의존성 설치

```sh
$> cargo build
```

## 1.3 서버 실행

```sh
$> cargo run
```
